# 🛒 Sistema de Estoque em Portugol

Projeto desenvolvido para treinar **lógica de programação** utilizando Portugol.

---

## 📌 Descrição

Este sistema simula o controle de estoque de um supermercado, permitindo cadastrar produtos, consultar informações e realizar operações básicas de gerenciamento.

O foco principal do projeto é o desenvolvimento da lógica de programação, sem uso de banco de dados ou interface gráfica.

---

## ⚙️ Funcionalidades

* 📦 Cadastro de produtos
* 📋 Listagem de produtos
* 🔍 Busca de produto por nome
* 🔄 Alteração de quantidade
* 💰 Identificação do produto mais caro
* 🧮 Cálculo do valor total do estoque
* 📊 Cálculo do valor total de um produto específico

---

## 🧠 Conceitos utilizados

* Vetores (arrays)
* Estruturas de repetição (`para`, `enquanto`, `repita`)
* Estruturas condicionais (`se`, `senao`)
* Procedimentos
* Busca linear
* Validação de dados
* Acumuladores
* Controle de fluxo com menu

---

## 🚀 Objetivo

Fortalecer a base em lógica de programação para evolução futura em linguagens como:

* Python
* C
